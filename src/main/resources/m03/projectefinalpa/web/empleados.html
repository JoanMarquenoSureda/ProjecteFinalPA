¡Visualiza los datos de los empleados!

Pasos a seguir para visualizar un empleado y sus horarios

En esta pantalla vas a poder visualizar los datos personales de un empleado y todos sus horarios asignados desde la fecha en que hagas la consulta.

1. Buscar Empleado

Para buscar un empleado, debes introducir el nombre completo del empleado que esté registrado en la base de datos en la celda "Nombre empleado". Si solo ingresas el nombre, el sistema no lo encontrará y aparecerá un mensaje indicando "No encontrado". Si el empleado no está registrado en la base de datos, también se mostrará el mismo mensaje. Una vez que hayas ingresado el nombre completo, puedes hacer clic en el botón "Buscar" para iniciar la búsqueda.


2. Visualizar sus datos personales

En esta sección podrás ver diferentes campos del empleado, como su nombre, dirección, número de teléfono, correo electrónico y una foto para poder identificarlo fácilmente.

3. Horarios del empleado

En la lista de horarios, podrás ver todos sus horarios asignados desde el momento de la búsqueda, junto con la zona asignada y una descripción que indica el nombre y la ubicación para acceder fácilmente. Los horarios están ordenados de manera cronológica, de más próximo a más lejano, lo que permite que el empleado tenga una planificación clara y organizada en el tiempo.

4. Eliminar asignación horarios

Con este botón podrás eliminar uno o varios horarios asignados al empleado buscado.

Para más ayuda, puedes ver el apartado "Conclusión", donde verás la parte más técnica de la aplicación y de como fue creada.
